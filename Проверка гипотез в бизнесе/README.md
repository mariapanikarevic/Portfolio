# Проверка гипотез в бизнесе
Сервис проката самокатов предоставил три датасета с данными о пользователях сервиса, их поездках за один год, а также стоимости услуг в подписках.

**Задача** - проанализировать демографию пользователей и особенности использования самокатов, а также определить возможную выгоду от распространения платной подписки на самокаты.

**Ход работы:**
- Загрузка данных и предварительная подготовка
- Исследовательский анализ данных (EDA)
- Объединение данных
- Подсчет выручки
- Проверка гипотез
- Распределения
